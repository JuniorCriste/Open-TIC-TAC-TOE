Open Tic Tac Toe

Relatório de Atualizações

Versão 0.1

  • Open Tic Tac Toe postado no Github com função Player x Player.
  • Interface simplista.

    • Versão 1

  • Atualizado com função Player x Player e Player x Computer
  • Game tem sons adicionados de clique e vitória.
  • Interface é atualizada.
  • Inclusão da tela de créditos.

    • Versão 1.01

  • Game tem mais sons adicionados indicando quem é o campeão da partida.
  • Interface é atualizada.
  • Adição de trilhas sonoras, onde o usuário pode escolher entre quatro diferentes músicas
  • Agora todos os cursores são customizados.
  • Inclusão da tela de configurações.

    • Versão 1.50

  • Codigo foi limpo e flexibilizado com funções automatizadas.
  • Adicionado tela de campeão ao invés de um simples "alertbox".
  • Estudos com fundo animado, usando movimento nos eixos X e Y de uma imagem grande, ainda em fase de testes e possívelmente não entrará nas próximas versões
  • Troca de quem começa em cada rodada

    • Versão 1.70

  • Adicionado função de histórico inteligente de dados referênte a partidas anteriores.
  • Criação de diretorios ocultos para armazenar dados de config.
  • Reconhecimento da primeira sessão jogável.
  • Ícone atualizado.
  • Adicionada tela com placar histórico.

    • Versão 2.0

  • Adicionado troca de background junto a música.
  • Adicionado mais um arquivo shell as configurações, para definição dos fundos.
  • Bug que permitia marcar uma área já marcada usando o teclado foi resolvido.
  • Adição da tela e links de Donate.
  • Atualização de Interface da tela de créditos.
  • Criação de pacote de instalação para versão 2.0 compativeis com Ubuntu, Mint, Debian, RedHat, OpenSuse, Slackware, Fedora, CentOS e baseados nessas distros.
  • Atualização do Readme e das capturas de tela na documentação do GitHub e do site.
